## Renk Kararları
- Primary: `#0A0A0F` (derin siyah-lacivert)
- Accent: `#3B82F6` (elektrik mavisi) → gradient ile `#8B5CF6` (mor)
- Neon: `#06B6D4` (cyan — hover/glow efektleri için)
- Surface: `#111118`
- Border: `rgba(255,255,255,0.08)`
- Text Primary: `#FAFAFA`
- Text Secondary: `#A1A1AA`

## Font Kararları
- Heading: **Geist** (Vercel'in fontu — teknolojik, modern)
- Body: **Inter** (okunabilirlik)
- Mono: **Geist Mono** (kod blokları için)
